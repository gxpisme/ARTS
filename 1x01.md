## Algorithm

###
https://leetcode.com/problems/defanging-an-ip-address/submissions/

这个题比较简单，总结来说就是字符串替换。

```
class Solution {
    public String defangIPaddr(String address) {
        String ret = "";
        if (address.length() < 1) {
            return ret;
        }

        for (int i = 0; i < address.length(); i++) {
            char c = address.charAt(i);
            if (c == '.') {
                ret += "[.]";
            } else {
                ret += c;
            }
        }

        return ret;
    }
}
```
