## Algorithm
https://leetcode.com/problems/di-string-match/submissions/
```java
class Solution {
    public int[] diStringMatch(String S) {
        int[] ret = new int[S.length() + 1];
        int retI = 0;
        int ii = 0;
        int dd = S.length();
        for (int i = 0; i < S.length(); i++) {
            ret[retI++] = S.charAt(i) == 'I' ? ii++ : dd-- ;
        }
        ret[retI++] = ii;
        return ret;
    }
}
```
